登录系统:

请求消息:
POST  /api/mgr/signin  HTTP/1.1
Content-Type:   application/x-www-form-urlencoded

请求参数:
http 请求消息 body 中 参数以 格式 x-www-form-urlencoded 存储

需要携带如下参数:
username 用户名
password 密码

响应消息:
HTTP/1.1 200 OK
Content-Type: application/json

响应内容:
http 响应消息 body 中， 数据以json格式存储，
如果登录成功，返回如下:
{
    "ret": 0
}

ret 为 0 表示登录成功

如果登录失败，返回失败的原因，示例如下:
{
    "ret": 1,    
    "msg":  "用户名或者密码错误"
}

ret 不为 0 表示登录失败， msg字段描述登录失败的原因

登陆校验：

如果校验成功，无反馈，直接进入当前页面
如果校验失败，返回错误号与失败原因，并跳转至登陆页面，示例如下：
{
    'ret': 302,
    'msg': '未登录',
    'redirect': '/usr/signin.html'
}

注册系统:

请求消息:
POST  /api/mgr/signin  HTTP/1.1
Content-Type:   application/x-www-form-urlencoded

请求参数:
http 请求消息 body 中 参数以 格式 x-www-form-urlencoded 存储

需要携带如下参数:
username 用户名
password 密码
password2 密码确认
email 邮箱

响应消息:
HTTP/1.1 200 OK
Content-Type: application/json

响应内容:
http 响应消息 body 中， 数据以json格式存储，
如果注册成功，返回如下:
{
    "ret": 0
}

ret 为 0 表示登录成功

如果登录失败，返回失败的原因，示例如下:
{
    "ret": 1,    
    "msg":  "两次密码不一致"
}

ret 不为 0 表示登录失败， msg字段描述登录失败的原因
